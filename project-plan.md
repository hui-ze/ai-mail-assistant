# Foxmail 邮件智能摘要和待办提取桌面应用 - 项目规划文档

## 项目概述
开发一个桌面应用程序，通过 IMAP 协议读取 Foxmail 邮件，集成 AI 能力对邮件进行智能摘要和待办事项提取，并可选同步回 Foxmail 或系统日历。

## 技术架构
- **前端**：PyQt5 桌面应用框架
- **后端**：Python + IMAP + AI 服务集成
- **数据存储**：SQLite 数据库
