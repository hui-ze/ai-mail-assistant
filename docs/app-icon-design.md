## 🎨 图标设计方案（3个选项）

### 方案A：智能信封（推荐）

#### 设计概念
将传统信封图标与AI元素融合，表达"智能邮件助手"的核心定位。

#### 视觉元素
```
┌─────────────────┐
│  ╔═══════════╗  │
│  ║  ✉️ 信封   ║  │  - 渐变紫色背景
│  ║   🔮 AI   ║  │  - 信封内部有神经网络图案
│  ║   ✓待办   ║  │  - 右下角待办提取徽章
│  ╚═══════════╝  │
└─────────────────┘
```

#### 详细规格
- **主形状**: 圆角正方形（border-radius: 22%）
- **背景**: 线性渐变 #667eea → #764ba2，135度角
- **信封图标**: 
  - 白色描边，线宽3px
  - 内部填充半透明白色（opacity: 0.15）
  - 信封口向下，表示"打开"状态
- **AI神经网络**:
  - 信封内部叠加3-4个连接节点
  - 节点用小圆点表示，连线细而优雅
  - 颜色：白色带渐变发光效果
- **待办徽章**:
  - 右下角小圆形，橙红色背景
  - 内含白色对勾符号
  - 尺寸：主图标的1/4大小

#### 配色方案
```
主色渐变: #667eea → #764ba2
辅助色:
  - 白色: #FFFFFF
  - 待办徽章: #FF6B6B
  - 阴影: rgba(102, 126, 234, 0.3)
```

#### 适用场景
- ✅ Windows任务栏
- ✅ 桌面快捷方式
- ✅ 应用窗口标题栏
- ✅ 系统托盘图标（需简化版）

### 方案B：抽象大脑邮件

#### 设计概念
用抽象的大脑轮廓融合邮件符号，强调AI智能处理能力。

#### 视觉元素
```
┌─────────────────┐
│  ╔═══════════╗  │
│  ║  🧠 大脑   ║  │  - 大脑轮廓作为主体
│  ║   ✉️ 邮件  ║  │  - 邮件符号嵌入大脑中心
│  ║   ✓✓✓    ║  │  - 多个待办标记散布
│  ╚═══════════╝  │
└─────────────────┘
```

#### 详细规格
- **背景**: 同方案A，紫色渐变
- **大脑轮廓**: 
  - 简化的侧视图
  - 白色线条，带艺术感
  - 内部有邮件符号重叠
- **邮件符号**: 位于大脑中心，半透明白色
- **待办标记**: 3个小对勾，散布在图标周围

#### 优点
- 更强调AI能力
- 视觉辨识度高
- 现代科技感强

#### 缺点
- 可能在小尺寸下细节丢失
- 不如信封直观

### 方案C：三栏布局抽象

#### 设计概念
将应用的三栏布局抽象化为图标，强化产品界面特征。

#### 视觉元素
```
┌─────────────────┐
│  ╔═══════════╗  │
│  ║ ▌  ▌  ▌  ║  │  - 三条竖线代表三栏布局
│  ║ ▌📧▌✓  ║  │  - 中间栏有邮件符号
│  ║ ▌  ▌📋  ║  │  - 右侧有待办图标
│  ╚═══════════╝  │
└─────────────────┘
```

#### 详细规格
- **背景**: 紫色渐变
- **三栏抽象**: 
  - 三条不等宽的白色半透明矩形
  - 中间栏最宽
  - 圆角设计
- **图标嵌入**: 
  - 左栏：邮件符号
  - 中栏：AI闪光符号
  - 右栏：待办清单

#### 优点
- 与应用界面呼应
- 独特性强
- 适合已有用户的品牌认知

#### 缺点
- 对新用户不够直观
- 小尺寸下识别困难

## 📐 技术规格

### 图标尺寸矩阵

| 用途 | 尺寸 | 文件名 |
|------|------|--------|
| 应用主图标 | 256x256 | app-icon-256.png |
| 桌面快捷方式 | 128x128 | app-icon-128.png |
| 任务栏 | 64x64 | app-icon-64.png |
| 系统托盘 | 32x32 | app-icon-32.png |
| 窗口标题栏 | 16x16 | app-icon-16.png |

### 文件格式要求
- **主格式**: PNG（支持透明背景）
- **备选格式**: ICO（Windows应用）、SVG（矢量版）
- **色彩模式**: RGBA
- **色彩深度**: 32位

### 输出文件结构
```
assets/
└── icons/
    ├── app-icon-16.png
    ├── app-icon-32.png
    ├── app-icon-64.png
    ├── app-icon-128.png
    ├── app-icon-256.png
    ├── app-icon.ico         # Windows应用图标
    └── app-icon.svg         # 矢量源文件
```
